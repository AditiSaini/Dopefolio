Project Overview

The goal of this project is to explore how research collaboration among Nanyang Technological University Computer Science faculty members led to the improvement in standards of the department, which is reflected in various university ranking metrics. Such collaboration is represented using using network graphs and metrics like-clustering coefficient, degree and eigenvector centrality, etc.
